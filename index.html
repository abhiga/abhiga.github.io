WebCrawler & SearchEngine | C, C++, FSM
Developed a WebCrawler application which parses TITLE and META tags from HTML pages. It then creates a hashmap of these words with the linked list of index of URLs stored. The SearchEngine application can then be used to run queries of words on the data parsed by WebCrawler. The above applications can be used together to search any public data on a Website.

Was developing a web application for 'A Fresh Start' (previously Soap & Towel Campaign) to track donations for Homeless shelters. I worked on that part of the web application where donors can signup, login and update their profile.

Safewalk Server & Safewalk Android application | Java
SafeWalk is a system used at Purdue to help people get home safely at night if they feel unsafe, or to call the police if there's been an emergency and they have no other way to contact the authorities. Our final project in CS180 is to implement a replica of this system using a server and an android app. On the app, users can select their location, and their required destination, or call for emergency help. This request goes to the server, and then gets sent as a notification to the nearest volunteer walker who will then go to the requester and escort them home. Emergency help dials the Purdue Police department.
